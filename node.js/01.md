## Node.js介绍

node.js.org

### Node.js是什么

- 不是一门语言

- 不是库、不是框架

- 是一个JavaScript运行时得环境

- 可以解析和执行JavaScript代码

- 以前只有浏览器可以解析执行JavaScript代码

- 现在的JavaScript可以完全脱离浏览器来运行，一切可以归功于Node.js

  #### 浏览器中的JavaScript

  - EcmaScript

    - 基本的语法
    - if
    - var
    - function
    - Object
    - Array

  - BOM

  - DOM

    #### Node.js中的JavaScript

    - 没有**BOM、DOM**

    - EcmaScript

    - 在Node这个JavaScript执行环境中为JavaScript提供了一些服务器级别的操作API

      - 例如文件读写

      - 网络服务的构建
      - 网络通信
      - http服务器
      - 等处理。。。

    #### Node.js特性

    - 事件驱动
    - 非阻塞IO模型（异步）
    - 轻量和高效
